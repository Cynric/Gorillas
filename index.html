Gorillas by lhunath

An iPhone (or iPod touch) port of the popular old QBasic game with spunk!

Gorillas is an iPhone game based on the gameplay of an old game written by IBM folks in the QBasic language, and first distributed by Microsoft in MS-DOS 5 as a demonstration of the programming language.

This iPhone port of the old game introduces some convenience features (such as one-tap throwing), improved graphics (no kidding), and competitive gameplay through the introduction of a dynamic AI and a scoring system.

Find out all about it and view a demonstration at the Gorillas Homepage or check out the source code at GitHub.

Dependencies

The official iPhone SDK, cocos2d-iphone

Install

  • Clone the repository
  • Obtain cocos2d-iphone
  • Open both in Xcode
  • For each red folders in the Gorillas project, drag the respective folder from cocos2d-iphone into the Gorillas project to replace the red folders.
